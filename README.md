TripNest – Full Stack MERN Application

TripNest is a full-stack web application developed using the MERN stack that enables users to explore, create, and manage trips.
The project demonstrates strong backend development skills, secure authentication, and frontend integration using modern web technologies.

⸻

Tech Stack

Frontend
	•	React.js
	•	React Router
	•	Axios
	•	CSS

Backend
	•	Node.js
	•	Express.js
	•	MongoDB
	•	Mongoose
	•	JWT (JSON Web Tokens)

Tools
	•	Postman (API Testing)
	•	Git & GitHub

⸻

Features

Authentication & Authorization
	•	User registration and login using JWT
	•	Secure token-based authentication
	•	Protected routes accessible only to authenticated users

User Features
	•	Create, view, update, and delete trips
	•	Authenticated users can manage their own trips
	•	RESTful API design following best practices

Admin Access
	•	Admin-specific APIs and protected routes
	•	Role-based access control implemented in the backend
	•	Admin interface in React is being progressively extended to support advanced management features

⸻

Backend Highlights
	•	Clean MVC architecture
	•	Secure authentication middleware
	•	MongoDB schema relationships using Mongoose
	•	All APIs tested thoroughly using Postman
	•	Authorization handled using JWT tokens
